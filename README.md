# Event Management System 🎯  
### BITcoders | Elite Hack 1.0

A centralized platform designed to simplify college event and hackathon management.  
It replaces manual tools like WhatsApp, Google Forms, and spreadsheets with a single system for managing registrations, teams, events, and attendance.

---

# 🚩 Problem
Managing events using multiple tools leads to:
- Data duplication
- Manual errors
- Poor coordination
- No real-time tracking
- Difficult team management
- Lack of automated notifications

Organizers struggle to monitor registrations, teams, and attendance efficiently.

---

# 💡 Solution
Our platform provides a **centralized event management system** where organizers can create, manage, and track events in real time.

Key improvements include:
- Automated registration tracking
- Smart team management
- QR-based event check-in
- Role-based access control
- Automated notifications

# 🏗 System Architecture

User → Web Application → Django Backend → Database → Authentication System

### Flow
1. User interacts with the web application.
2. Frontend sends requests to Django backend.
3. Backend processes event logic (teams, registrations, QR generation).
4. Data is stored and retrieved from the database.
5. Authentication ensures role-based access for hosts and participants.

---

# 🗄 Database
The database stores:

- User data  
- Event information  
- Team details  
- Registration records  
- QR passes  
- Attendance records  

Ensures **secure storage and data integrity**.
